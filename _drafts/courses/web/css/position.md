1. static，默认值，元素无设置的时候就是static
2. relative，相对定位，对于当前位置进行调整
3. absolute，绝对定位，对于非static的父节点进行定位
4. fixed，浏览器绝对定位，对于浏览器进行定位
5. sticky，粘性定位，介于realtive与fixed之间，top与滚动配合，滚前是relative，滚后是fixed
