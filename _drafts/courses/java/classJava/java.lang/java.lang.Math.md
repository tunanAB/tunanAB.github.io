### @常用方法

```java
public static double ceil(double a);//返回大于或等于传入参数的最小整数。具体而言，如果参数是正数，则返回大于或等于该参数的最小整数，如果参数是负数，则返回小于或等于该参数的最小整数。
public static int round(float a);//将浮点数结果四舍五入到最接近的整数。
public static double pow(double base, double exponent);//计算幂函数
public static double abs(double a);//计算绝对值
```
